# Jorvea SEO Super Guide

This guide helps your site rank better for searches like **"how to earn money from jorvea"** and related creator queries.

## 1) Technical SEO (already implemented)

- Canonical + metadata base domain set to `https://jorvea.tech`
- `sitemap.xml` and `robots.txt` aligned to the same domain
- Favicon + app icons configured
- Organization + Website JSON-LD added globally
- FAQ JSON-LD added to earning page

## 2) Must-do setup outside code

- Add and verify site in **Google Search Console**
- Add and verify in **Bing Webmaster Tools**
- Submit sitemap URL: `https://jorvea.tech/sitemap.xml`
- Request indexing for:
  - `https://jorvea.tech/`
  - `https://jorvea.tech/earning`
  - `https://jorvea.tech/support`
  - `https://jorvea.tech/contact`

## 3) Content SEO to rank "how to earn money from jorvea"

- Keep exact phrase in earning page:
  - Title
  - H1 or intro paragraph
  - Meta description
- Add one dedicated FAQ section on earning page with questions like:
  - How to earn money from Jorvea?
  - How payouts are calculated?
  - How to appeal earning disqualification?
- Update earning page at least every 2-4 weeks (freshness helps)

## 4) Internal linking strategy

- Link to `/earning` from:
  - Home hero section
  - Support page
  - Footer quick links
- Use anchor text like:
  - "How to earn money from Jorvea"
  - "Jorvea earning policy"
  - "Submit earning appeal"

## 5) On-page SEO checklist for every page

- One clear `h1`
- Unique title + description
- Canonical URL set
- Descriptive headings (`h2`, `h3`) with real keywords
- Natural keyword usage (avoid stuffing)
- All images have useful `alt` text
- Add at least one internal link to related page

## 6) Performance + UX SEO (Core Web Vitals)

- Keep hero images optimized (`next/image`)
- Avoid very large uncompressed PNG/JPG files
- Keep animation subtle and lightweight
- Check on mobile for layout shift and tap targets

## 7) Off-page SEO

- Get backlinks from:
  - Your official social pages
  - Product listing pages
  - Creator communities
- Keep Name/Brand consistency: `Jorvea` everywhere

## 8) Weekly SEO workflow

1. Check Search Console performance (queries + pages)
2. Find pages with high impressions but low CTR
3. Improve title/meta on those pages
4. Add 1-2 internal links to `/earning`
5. Re-submit updated page URL for indexing

## 9) Suggested target keywords

- how to earn money from jorvea
- jorvea earning policy
- jorvea monetization guide
- jorvea payout policy
- jorvea appeal process

## 10) Common mistakes to avoid

- Different domains in canonical/sitemap/robots
- Duplicate page titles
- Thin pages with no useful content
- Missing alt text on brand and feature images
- No clear path from home page to earning page
